## ✍️ Commands and Usage  

### 📜 Viewing the Task List  
**Command:**  
```sh
list
```
**Expected Output:**  
```sh
No tasks in your list, mortal.
```

---

### 📌 Adding a Task  
**Command:**  
```sh
todo <task_name>
```
**Example:**  
```sh
todo Add Ui.png
```
**Expected Output:**  
```sh
Added: [T] [ ] Add Ui.png
```

---

### ✅ Marking a Task as Done  
**Command:**  
```sh
mark <task_number>
```
**Example:**  
```sh
mark 1
```
**Expected Output:**  
```sh
Marked as done: [T] [X] Add Ui.png
```
